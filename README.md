# PetusMod (petus.gdps)

Мод **Geode** для клиента Geometry Dash 2.2, превращающий его в клиент
**PetusGDPS**. Ставится в сборку игры, которую раздаёт PetusLauncher.

## Что делает
- **Авторизация без пароля**: читает сессию лаунчера
  (`%LOCALAPPDATA%\PetusGDPS\session.json`) и подставляет её в игру. Токен
  уходит на ядро как `m_GJP2 = "petus:<token>"`, поэтому статистика, рекорды и
  профиль сохраняются на реальный аккаунт.
- **Текстбоксы**: «не залогинен — войди через лаунчер», приветствие + бонус
  новичку, уведомление о бане (причина + время).
- **Проверка версии**: если установленная сборка старее серверной — предлагает
  обновиться через лаунчер.
- **Своя вкладка Mods** в Options (список модов + их настройки); кнопка Geode из
  главного меню убрана.
- **Кастомные Play-уровни**: главные уровни заменяются на заданные через
  `/api/defaultlevels`.
- **Скриншот-превью** на 50% прохождения → imgbb → превью уровня на сайте.
- **Deep link** `petusgdps://play?level=<id>` (кнопка Play на сайте).

## Настройки (mod.json)
- `server-base` — база API (по умолчанию `https://gdps.petus.ru`).
- `disable-ingame-login` — прятать кнопку аккаунта в меню.
- `imgbb-key` — ключ для загрузки превью.

## Сборка
Нужны Geode SDK v5.8.0 (GD 2.2081), CMake, Ninja, **clang-cl** (обычный MSVC
`cl.exe` падает на заголовках Geode).

```bat
build-clang.bat
```
Результат: `build/petus.gdps.geode` (авто-устанавливается в игру).

Зависимости: `geode.node-ids`. В боевой сборке рядом стоят GMD/Editor API,
QoL-моды и текстуры — их список формирует PetusLauncher.
